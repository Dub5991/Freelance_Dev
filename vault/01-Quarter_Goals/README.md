## Purpose
Strategic goals and OKRs (Objectives and Key Results) for each quarter.

## Structure
Create one file per quarter: `Q1-2026.md`, `Q2-2026.md`, etc.

## What Goes Here
- **Revenue Goals** - Target income for the quarter
- **Client Goals** - New clients to acquire, contracts to renew
- **Skill Goals** - Skills to develop or certify
- **Business Development** - Partnerships, marketing initiatives
- **Personal Development** - Learning, networking, health

## Template Structure
```markdown
# Q1 2026 Goals

## Revenue Target: $45,000

### Objective 1: Grow Revenue
- [ ] KR1: Sign 2 new clients at $150/hr
- [ ] KR2: Increase billing by 20% with existing clients
- [ ] KR3: Close $50K in new contracts

### Objective 2: Expand Expertise
- [ ] KR1: Complete AWS Solutions Architect certification
- [ ] KR2: Build 3 AI integration projects for portfolio
- [ ] KR3: Write 2 technical blog posts

### Objective 3: Strengthen Relationships
- [ ] KR1: Have quarterly check-ins with all active clients
- [ ] KR2: Attend 2 industry events
- [ ] KR3: Make 5 warm introductions
```

## Review Cadence
- **Monthly** - Check progress on KRs
- **End of Quarter** - Full retrospective and next quarter planning
- **Yearly** - Annual review incorporating all quarters

## MCP Integration
Use **llc_ops_server** to track actual revenue vs. targets

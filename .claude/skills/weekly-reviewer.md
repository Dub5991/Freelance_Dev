# Weekly Reviewer Skill

## Description
Conduct comprehensive weekly business reviews to assess progress, identify issues, and plan ahead effectively.

## When to Use This Skill
- Every Friday afternoon or Monday morning
- Before client status updates
- When planning the upcoming week
- During monthly/quarterly reviews
- When assessing business health

## MCP Servers Used
- **work_server**: Task and project status
- **client_server**: Client relationship health
- **billing_server**: Financial performance
- **career_server**: Professional development progress

## Weekly Review Framework

### Step 1: Review Past Week (15 min)

**Tasks & Projects:**
```
1. List completed tasks using list_tasks()
2. Review hours logged by client
3. Check project progress
4. Identify blockers or delays
5. Celebrate wins
```

**Example Prompts:**
- "Show all tasks completed this week"
- "How many billable hours did I log this week?"
- "What's the status of all active projects?"

**Client Communications:**
```
1. Review client interactions
2. Check health scores
3. Note any concerns
4. Plan follow-ups
```

**Example Prompts:**
- "Show communications with clients this week"
- "Calculate health scores for all active clients"

**Financial Review:**
```
1. Review invoices sent/paid
2. Check outstanding receivables
3. Track expenses
4. Calculate weekly revenue
```

**Example Prompts:**
- "Show invoices sent this week"
- "What revenue came in this week?"
- "List this week's business expenses"

### Step 2: Assess Current State (10 min)

**Active Work:**
```
- Open P0 tasks (max 3)
- Tasks by priority
- Projects at risk
- Upcoming deadlines
- Resource allocation
```

**Example Prompts:**
- "List all P0 tasks"
- "Show projects due in next 2 weeks"
- "Are any clients at risk?"

**Business Health:**
```
- Client health scores
- Cash flow status
- Overdue invoices
- Pipeline visibility
```

**Example Prompts:**
- "Show clients with health scores below 70"
- "List overdue invoices"

### Step 3: Plan Next Week (15 min)

**Priority Setting:**
```
1. Review upcoming deadlines
2. Identify critical tasks
3. Allocate time blocks
4. Schedule client meetings
5. Plan learning time
```

**Example Prompts:**
- "What are my top priorities for next week?"
- "Show upcoming meetings and deadlines"

**Proactive Actions:**
```
- Send client updates
- Follow up on overdue payments
- Schedule important calls
- Block focus time
- Plan learning activities
```

### Step 4: Document Insights (5 min)

**Record:**
- Key achievements
- Lessons learned
- Risks identified
- Decisions made
- Action items

## Monthly Deep Dive

### Do Monthly (First Week)
```
1. Full financial review
2. Client portfolio analysis
3. Skills and learning review
4. Rate and positioning check
5. Tax planning update
6. Business process review
```

**Example Prompts:**
- "Generate monthly revenue report"
- "Calculate profit margin for the month"
- "Show all learning goals progress"
- "List completed vs active clients"

## Quarterly Strategy Review

### Do Quarterly
```
1. Strategic goal review
2. Market positioning assessment
3. Service offering evaluation
4. Pricing strategy review
5. Client mix analysis
6. Professional development plan
```

**Example Prompts:**
- "Show quarterly revenue by client"
- "Calculate quarterly tax estimate"
- "Review career milestones"

## Time Management

### Weekly Time Allocation
- **Billable Work**: 60-70%
- **Business Operations**: 15-20%
- **Marketing/Sales**: 5-10%
- **Learning**: 5-10%
- **Admin**: 5-10%

### Ideal Week Structure
```
Monday: Planning & admin
Tuesday-Thursday: Deep work on projects
Friday: Client meetings, weekly review
Weekend: Rest and recharge
```

## Review Cadence

| Frequency | Focus | Duration |
|-----------|-------|----------|
| Daily | Priorities and progress | 5 min |
| Weekly | Detailed business review | 45 min |
| Monthly | Financial and strategic | 2 hours |
| Quarterly | Big picture and goals | 4 hours |
| Annual | Comprehensive planning | 1 day |

## Success Patterns

### What Good Looks Like:
- Consistent billable hours
- High client satisfaction
- Growing revenue
- Healthy profit margins
- Regular learning
- Balanced workload
- Clear pipeline

### Warning Patterns:
- Feast or famine cycles
- Client churn
- Revenue decline
- Margin erosion
- No learning time
- Constant stress
- Empty pipeline

## Action Planning

### When Things Are Good:
- Maintain momentum
- Build reserves
- Invest in growth
- Raise rates
- Expand services

### When Things Are Slow:
- Increase marketing
- Reach out to network
- Update portfolio
- Learn new skills
- Optimize processes

### When Overwhelmed:
- Prioritize ruthlessly
- Delegate or outsource
- Raise rates (reduce volume)
- Improve processes
- Say no more often

## Example Workflows

### Friday Afternoon Review
```
1. Export week's data from all servers
2. Calculate key metrics
3. Fill out review template
4. Identify top 3 priorities
5. Schedule next week's time blocks
6. Send client updates if needed
7. Document lessons learned
```

### Monday Morning Planning
```
1. Review last week's summary
2. Check calendar for week
3. Confirm top priorities
4. Block focus time
5. Send good morning to clients
6. Start on P0 tasks
```
